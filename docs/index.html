Architecture

Unified binary: SOCAT 1.7.3.3 with integrated argument hiding and process masquerading built-in.

1

Process Masquerading

Set process name via prctl (Linux), setproctitle (BSD). Options: -Ms (systemd), -MS (sshd), -Mn (NetworkManager), -Md (dbus), -Mr (resolved), -Mc (custom).

2

PID Targeting

Manipulate /proc/sys/kernel/ns_last_pid to receive specific PIDs. Blend into low PID ranges. Mimic service restarts. Option: -Mp <pid>

3

Time Namespace Matching

Match target process start time using time namespaces (kernel 5.6+). Process appears with identical start time in ps. Option: -Mt <pid>

4

Operational Hardening

OOM immunity (-Mo), ephemeral port range control (-MP), environment sanitization (-Me). Survive memory pressure and remove forensic artifacts.

Getting Started

Build from source and deploy. Multi-platform support with native optimizations.

bash 
$ # Clone repository
git clone https://github.com/Real-Fruit-Snacks/Conduit.git
cd Conduit

$ # Build all components
make

$ # Test installation
make test

$ # Verify masquerade options
./conduit --list-masq

Prerequisites

  • GCC or compatible C compiler
  • GNU Make for build system
  • OpenSSL (optional, for SSL/TLS)
  • GNU Readline (optional, for interactive features)

Platform Support

  • Linux (Ubuntu, Debian, RHEL, Alpine)
  • FreeBSD, OpenBSD, NetBSD
  • macOS (limited masquerading)
  • Solaris, AIX (generic fallback)

Usage Examples

Multiple deployment patterns for different operational scenarios.

basic-masquerade.sh 
# Masquerade as systemd service
./conduit -Ms \
  TCP-LISTEN:8080,fork \
  TCP:backend:80

# Process appears as systemd-logind
pid-targeting.sh 
# Target specific PID (requires root)
sudo ./conduit -Ms -Mp 500 \
  TCP-LISTEN:8080 \
  TCP:target:80

# Process receives PID 500
time-matching.sh 
# Match target process start time
sudo ./conduit -Ms -Mt 500 \
  TCP-LISTEN:8080 \
  TCP:target:80

# Same start time as PID 500
full-stealth.sh 
# Full stealth: PID + time + OOM + env clean
sudo ./conduit -Ms -Mp 500 -Mt 500 \
  -Mo -Me -MP 49152-65535 \
  TCP-LISTEN:8080 TCP:target:80

# Maximum operational security
