Features

🔒

Argument Hiding

Command-line arguments automatically hidden from process inspection tools (ps, top, htop, /proc/cmdline). Zero configuration required.

🎭

Process Masquerading

Appear as legitimate system processes: kernel workers, systemd services, SSH daemons, or custom names. Multiple preset identities included.

🌐

Full SOCAT Power

All original SOCAT capabilities: TCP, UDP, UNIX sockets, SSL/TLS, SOCKS, HTTP CONNECT, and 50+ data channel types with 100+ configuration options.

💻

Cross-Platform

Native support for Linux (prctl), BSD (setproctitle), and macOS. Generic fallback for other Unix-like systems. Tested on multiple distributions.

📦

Multiple Deployments

Choose your deployment: unified Conduit binary with presets, standalone wrapper, or modified SOCAT with integrated stealth library.

⚡

Zero Overhead

Minimal performance impact. Stealth features activate in microseconds. No changes to SOCAT's core relay logic or data flow.

Quick Start

# Clone repository
git clone https://github.com/Real-Fruit-Snacks/Conduit.git
cd Conduit

# Build all components
make

# Test installation
make test

# Masquerade as kernel worker
./conduit --masq-kernel TCP-LISTEN:8080,fork TCP:10.0.0.5:80

# List available masquerade options
./conduit --list-masq
